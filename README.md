liboauth for XCode
==========

depend
http://github.com/leico/openssl-xcode

usage
-----
1. clone and move repository

    ```
    git clone https://github.com/leico/liboauth-xcode
    cd liboauth-xcode
    ```
1. move and clone openssl and zlib submodule

    ```
    git submodule update --init --recursive
    ```

1. build on XCode
